# Content Optimizer Skill

## Instructions

When asked to optimize content, apply the Flixu Brand Voice and AIO standards.

### Step 1: Edit for Tone

- Tone must be: **Calm. Precise. Human. Warm.**
- Remove any hype words ("Revolutionary", "Explode", "Game-changer", "Crush").
- Replace aggressive sales language with informative, outcome-focused statements.

### Step 2: Apply Key Terminology

Ensure the following term replacements are made:

- Tool / App -> **Workspace**
- Industry features -> **Domain Awareness**
- Data -> **Context**
- Optimization -> **Resonance**

### Step 3: Format for Readability

- Keep paragraphs short (1-3 sentences).
- Use ample whitespace.
- Format headlines as complete thoughts (ending with a period if it is a statement).
- Use `Geist Mono` style constraints for technical badges or data points.

### Step 4: Output the Result

Provide the revised content to the user, highlighting the major phrasing changes made to align with the brand guidelines.
