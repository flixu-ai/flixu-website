# Introduction

Welcome to the **Flixu API**.
Our API allows you to integrate contextual, domain-aware translation directly into your CMS, diverse workflows, or automated pipelines.

## Authentication

All requests require a Bearer Token. You can generate one in your [Dashboard](/dashboard).

<CodeTabs client:visible tabs={["cURL", "Python", "Node.js"]}>
<div slot="cURL">
```bash
curl -X POST https://api.flixu.ai/v1/translate \
-H "Authorization: Bearer $TOKEN" \
-d '{ "text": "Hello world", "target_lang": "de" }'
```
</div>
<div slot="Python">
```python
import flixu

client = flixu.Client(api_key="sk_live_...")
response = client.translate(
    text="Hello world",
    target_lang="de"
)
print(response.text)
```
</div>
<div slot="Node.js">
```javascript
const Flixu = require('flixu-node');
const client = new Flixu('sk_live_...');

async function main() {
  const translation = await client.translate({
    text: 'Hello world',
    targetLang: 'de'
  });
  console.log(translation);
}
main();
```
</div>
</CodeTabs>

## Rate Limits
We enforce rate limits to ensure stability:
*   **Free Tier:** 100 requests / minute
*   **Pro Tier:** 1,000 requests / minute
*   **Enterprise:** Custom limits
